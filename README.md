# XLeRobot 家庭服务机器人 — 项目展示页

单文件静态网页（`index.html`，无构建、无外部依赖），展示 XLeRobot
家庭服务机器人项目的技术细节与成果：ROS 2 Jazzy 系统架构、标定体系、
传统 + 学习（ACT）混合抓取、激光建图与导航、交付工程化。

目标岗位：机器人系统 / ROS 2 应用工程师、具身智能应用工程师。

## 本地预览

```bash
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000
```

直接双击 `index.html` 也可以。

## 仓库结构

- `index.html` — 展示页全部内容（HTML + CSS + 内联 SVG 架构图）
- `assets/` — 页面引用的图片（整机渲染图、三个演示视频封面）

## 说明

- 页面中的三个演示视频链接至 B 站；源码随教学版开源前暂不公开。
- 页面中的数字与阈值均来自真实实现（标定质量门、ACT 训练基线、
  看门狗参数等），修改前请与实现核对。
